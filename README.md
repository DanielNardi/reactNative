*Tecnologia usada:
   *React Native: Framework para desenvolvimento mobile multiplataforma.

*Planejamento
   interface: criar styles  e subcomponente de botão para não ter códigos repetidos

   Processamento: Só é processado quando clica no botão "=" que utiliza a função eval()
